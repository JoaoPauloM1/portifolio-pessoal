## 🚀 Sobre o Projeto

Este projeto é um **Portfólio Online Pessoal**, desenvolvido como requisito para a Atividade Prática da disciplina de **Fundamentos da Programação Web** da Uninter. 

O site foi construído utilizando o conceito de **Single Page Website** com navegação fluida por âncoras, layout responsivo e um visual moderno no estilo *Dark Tech*.

---

## 🛠️ Tecnologias Utilizadas

Para cumprir as diretrizes acadêmicas, o projeto foi desenvolvido **do zero, sem o uso de frameworks** ou bibliotecas externas (como Bootstrap ou React):

* **HTML5 Semântico:** Estrutura limpa, hierárquica e acessível.
* **CSS3 Puro:** Variáveis nativas, Grid Layout, Flexbox e efeitos de vidro (*Glassmorphism*).
* **JavaScript Vanilla:** Manipulação do DOM nativa para interações e validação de formulários.

---

## 📌 Funcionalidades e Seções Obrigatórias

* ✅ **Menu Fixo:** Navegação por âncoras fluida que não exige recarregamento da página.
* ✅ **Sobre Mim:** Apresentação pessoal e hobbies.
* ✅ **Formação:** Histórico educacional e cursos.
* ✅ **Portfólio:** Galeria de trabalhos e links.
